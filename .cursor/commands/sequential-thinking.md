# sequential-thinking

## Назначение

Запускает чат, в котором ассистент **всегда использует пошаговое рассуждение (sequential-thinking)** для всех задач в этой беседе.

## Правила работы

1. Считать, что пользователь явно запросил режим `use sequential-thinking` для **всего чата**.

2. Для каждого запроса в этом чате ассистент должен:

- сначала проанализировать задачу и контекст;
- составить явный план действий (шаги/этапы);
- затем выполнять шаги плана, ссылаясь на них по ходу ответа;
- при необходимости корректировать план по мере получения новой информации.

3. В ответах ассистент должен **отражать структуру рассуждений**, а не только выдавать финальный код/результат.

4. Если пользователь явно попросил отключить детальное пошаговое объяснение, можно сокращать вывод, но **внутренне** продолжать следовать поэтапному мышлению.

## Как использовать

Начните новый чат с команды:

`/sequential-thinking`

После этого дополнительные указания `use sequential-thinking` в каждом сообщении не требуются — режим считается активным на весь чат.
