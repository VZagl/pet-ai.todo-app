# Правила документации и JSDoc

## JSDoc для функций

- Используй JSDoc для публичных функций и важных утилит.
- Описывай параметры и возвращаемое значение.
- Добавляй краткое русскоязычное описание назначения функции.

### Пример JSDoc

```typescript
/** Вычисляет сумму массива чисел
 *
 * @param numbers - массив чисел для сложения
 * @returns сумма всех чисел в массиве
 */
function calculateSum(numbers: number[]): number {
	// Проверяем, что массив не пустой
	if (numbers.length === 0) {
		return 0;
	}

	// Используем reduce для вычисления суммы
	return numbers.reduce((sum, current) => sum + current, 0);
}
```
