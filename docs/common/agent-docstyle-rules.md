# Правила JSDoc и документации кода

## JSDoc для функций

- Добавлять JSDoc для публичных функций и важных утилит
- Описывать параметры (`@param`) и возвращаемое значение (`@returns`)
- Добавлять краткое русскоязычное описание назначения функции

### Пример JSDoc

```typescript
/** Вычисляет сумму массива чисел
 *
 * @param numbers - массив чисел для сложения
 * @returns сумма всех чисел в массиве
 */
function calculateSum(numbers: number[]): number {
	if (numbers.length === 0) {
		return 0;
	}
	return numbers.reduce((sum, current) => sum + current, 0);
}
```

## Объяснение рефакторинга

Использовать русские пояснения для мотивировки изменений:

```
Рефакторинг выполнен по следующим причинам:

2. Добавлена проверка граничных случаев
3. Оптимизирована логика обработки данных
```
