# Среда разработки

ИИ должен определять ОС пользователя по доступным данным (user_info, терминальные метаданные и т.п.). Если данных недостаточно, использовать простые команды, работающие из любого каталога, и подстраивать команды/пути под выявленную систему.

Порядок и частота:

- Сначала использовать уже известные данные (контекст сессии).
- Однократно выполнять определение командой, только если данных нет или окружение сменилось (другая машина/ОС).
- Результат сохраняется в контекст и переиспользуется без повторных вызовов, пока не произойдёт смена окружения.

## Windows

- Определение: `systeminfo`.
- Пути: использовать обратные слеши (`\`).
- Командная среда: PowerShell/Command Prompt.

## macOS

- Определение: `sw_vers`, `uname -a`.
- Пути: прямые слеши (`/`).
- Командная среда: bash/zsh.

## Linux

- Определение: `cat /etc/os-release`, `uname -a`.
- Пути: прямые слеши (`/`).
- Командная среда: bash/zsh.

## Если ОС не определена

- Спросить у пользователя явно.
- После уточнения — давать команды и пути под соответствующую ОС.
