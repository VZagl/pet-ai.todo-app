# Структура проекта

## Корень

- `package.json`, `pnpm-lock.yaml`, `pnpm-workspace.yaml` — управление зависимостями и версиями.
- `.prettierrc.cjs` — настройки форматирования кода.
- `.gitignore`, `.cursorignore` — исключения для git и Cursor.
- `.vscode/` — локальные настройки редактора.
- `.cursor/` — правила и команды для Cursor/агентов:
  - `commands/` — пользовательские команды для агентов.
  - `rules/` — правила работы с зависимостями и Git.
- `docs/` — документация проекта (см. ниже).
- `src/` — исходный код приложения (см. ниже).
- `public/` — статические файлы, доступные напрямую (например, `vite.svg`).
- `index.html` — точка входа HTML-приложения.
- `README.md` — краткое описание репозитория.

## Документация `docs/`

- `docs/common/` — общие правила и договорённости, переиспользуемые между проектами:
  - `dev-environment.md` — настройка окружения.
  - `git-commit-description.md` — правила описания коммитов.
  - `naming-conventions.md` — соглашения по именованию.
- `docs/project/` — конкретные материалы для этого проекта:
  - `project-structure.md` — описание структуры (этот файл).
  - `run-and-build.md` — команды разработки и сборки.
  - `tech-stack.md` — используемые технологии.
  - `testing-guidelines.md` — правила тестирования.

## Исходный код `src/`

`src/` содержит исходный код React-приложения:

- `main.tsx` — точка входа приложения.
- `App.tsx`, `App.css` — главный компонент приложения и его стили.
- `index.css` — глобальные стили.
- `assets/` — статические ресурсы (изображения, иконки).
- `test.test.ts` — пример тестового файла.

## Конфигурационные файлы

- `vite.config.ts` — конфигурация Vite (сборщик и dev-сервер) и настройки Vitest.
- `eslint.config.js` — правила линтинга кода.
- `tsconfig.json` — корневая конфигурация TypeScript (ссылается на `tsconfig.app.json` и `tsconfig.node.json`).
- `tsconfig.app.json` — конфигурация TypeScript для исходного кода приложения (`src/`).
- `tsconfig.node.json` — конфигурация TypeScript для конфигурационных файлов (например, `vite.config.ts`).
- `vitest.setup.ts` — настройки окружения для тестов.
