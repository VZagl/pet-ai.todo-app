# Разработка

## Установка зависимостей

```bash
pnpm install
```

> **Примечание**: Проект использует `pnpm` как менеджер пакетов. При попытке использовать другие менеджеры (npm, yarn) будет автоматически вызван `preinstall` hook, который заблокирует установку.

## Запуск dev-сервера

```bash
pnpm dev
```

Запускает Vite dev-сервер с hot module replacement (HMR). По умолчанию доступен на `http://localhost:5173`.

## Сборка проекта

```bash
pnpm build
```

Выполняет:

1. Компиляцию TypeScript (`tsc -b`)
2. Сборку проекта через Vite (`vite build`)

Результат сборки сохраняется в директории `dist/`.

## Линтинг

```bash
pnpm lint
```

Запускает ESLint для проверки кода на соответствие правилам проекта.

## Превью сборки

```bash
pnpm preview
```

Запускает локальный сервер для предпросмотра собранного проекта (из директории `dist/`). Полезно для проверки production-сборки перед деплоем.
