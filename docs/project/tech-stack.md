# Технологии

## Фреймворк и библиотеки

- **React**: `^19.2.3` — библиотека для создания пользовательских интерфейсов.
- **React DOM**: `^19.2.3` — рендеринг React-компонентов в DOM.

## Язык программирования

- **TypeScript**: `~5.9.3` — типизированный надмножество JavaScript.

## Сборщик и инструменты разработки

- **Vite**: `^7.3.0` — быстрый сборщик и dev-сервер.
- **@vitejs/plugin-react-swc**: `^4.2.2` — плагин Vite для React с использованием SWC (быстрая компиляция).

## Линтинг и форматирование

- **ESLint**: `^9.39.2` — линтер для JavaScript/TypeScript.
  - **@eslint/js**: `^9.39.2` — базовая конфигурация ESLint.
  - **typescript-eslint**: `^8.50.1` — поддержка TypeScript в ESLint.
  - **eslint-plugin-react-hooks**: `^7.0.1` — правила для React Hooks.
  - **eslint-plugin-react-refresh**: `^0.4.26` — поддержка Fast Refresh в Vite.
  - **globals**: `^16.5.0` — глобальные переменные для ESLint.
- **Prettier**: `^3.7.4` — форматировщик кода.
  - **prettier-plugin-css-order**: `^2.1.2` — сортировка CSS-свойств.
  - **prettier-plugin-organize-attributes**: `^1.0.0` — организация атрибутов в HTML/JSX.
  - **prettier-plugin-organize-imports**: `^4.3.0` — организация импортов.
