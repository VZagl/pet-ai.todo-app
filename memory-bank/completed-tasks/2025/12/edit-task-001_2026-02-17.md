# edit-task-001

- **Название:** Редактирование текста задачи
- **Дата создания:** 2025-12-31
- **Дата завершения:** 2026-02-17
- **Уровень сложности:** Level 2 — Simple Enhancement
- **Тип:** Feature

## Задание

Inline редактирование по двойному клику. Валидация при редактировании. Отмена изменений по Escape. Связано с todo-app-001. Причина: улучшение UX — пользователи могут исправлять опечатки.

## Изменения в задании

- **Вход в режим редактирования:** вместо двойного клика по тексту — кнопка редактирования (иконка карандаша) слева от кнопки удаления. Причина: избежание случайного входа при двойном клике по тексту; явный intent пользователя.
- **Явные кнопки Save/Cancel:** добавлены кнопки «Сохранить» (✓) и «Отмена» (×) с иконками и цветным фоном — помимо Enter/Escape для сохранения/отмены.

## Результат

Реализовано inline-редактирование текста задачи в компоненте TodoItem. Единственный способ входа — кнопка редактирования (иконка карандаша) слева от кнопки удаления. Валидация: пустая строка и MAX_TODO_LENGTH. Enter — сохранить, Escape — отменить. Кнопки «Сохранить» (✓) и «Отмена» (×) с иконками и цветным фоном. Задержка 400 ms для action buttons после выхода — без layout shift. 15 тестов TodoItem проходят, сборка успешна.
