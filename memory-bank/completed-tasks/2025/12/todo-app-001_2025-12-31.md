## Задание

Полнофункциональное TO-DO приложение на React с использованием TypeScript, включающее полный CRUD функционал для управления задачами, фильтрацию, персистентность данных через localStorage и современный минималистичный UI.

Цель — создать pet-проект для демонстрации навыков работы с React 19, TypeScript (strict mode), custom hooks, comprehensive тестированием (Vitest + Testing Library) и современным UI/UX дизайном с accessibility.
