# i18n-001

- **Название:** Мультиязычность (i18n) — Русский и Английский
- **Дата создания:** 2026-01-13
- **Дата завершения:** 2026-02-16
- **Уровень сложности:** Level 3 — Feature
- **Тип:** Feature (Internationalization)

## Задание

- Установка и настройка `react-i18next` и `i18next`
- Создание структуры локалей: `src/locales/{ru,en}/translation.json`
- Настройка i18n конфигурации: `src/i18n/config.ts`
- Перевод всех текстовых элементов:
  - Плейсхолдеры и кнопки в компонентах
  - Сообщения об ошибках валидации
  - Счетчик задач и фильтры
  - ARIA-метки для accessibility
- Реализация функций плюрализации (склонения числительных):
  - Русский: задача/задачи/задач
  - Английский: task/tasks
- Добавление переключателя языка в UI (header/footer)
- Сохранение выбранного языка в localStorage
- Автоопределение языка браузера при первом запуске
- TypeScript типизация для ключей переводов

## Результат

Реализована полная мультиязычность приложения pet.todo: i18next + react-i18next, локали ru/en, плюрализация для счётчика задач, переключатель языка (кнопка с иконкой глобуса, dropdown) в правом верхнем углу. Язык сохраняется в localStorage, определяется по браузеру при первом запуске. Обновлены все компоненты под `t()`, создан `docs/project/i18n-guidelines.md`.

**Метрики:** ~8 файлов создано, ~12 изменено; Unit-тестов 119 (в т.ч. 3 LanguageSwitcher), E2E 18. Соответствие требованиям 100%.
