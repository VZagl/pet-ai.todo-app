- **Название:** TodoFilter: Оптимизация повторной фильтрации

- **Дата завершения:** 2026-02-04

## Задание

При клике на активный фильтр происходила повторная фильтрация/обновление состояния, хотя результат не менялся.

## Результат

- Активная кнопка фильтра отключена (`disabled`) и не вызывает смену фильтра при клике
- Добавлен guard в setter фильтра: обновление только при изменении значения
- Обновлены тесты под disabled-состояние и keyboard-навигацию
- Тесты `TodoFilter.test.tsx`: 17/17
- Тесты `use-filter.test.ts`: 7/7
- Регрессий нет
