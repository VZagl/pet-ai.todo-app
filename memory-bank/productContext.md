# Memory Bank: Product Context

## Product Vision

Современное, интуитивное TO-DO приложение, демонстрирующее лучшие практики React разработки.

## Target Users

- Разработчики, изучающие React
- Пользователи, нуждающиеся в простом инструменте управления задачами

## User Stories

### US-001: Добавление задачи

**Как** пользователь  
**Я хочу** добавлять новые задачи  
**Чтобы** отслеживать, что мне нужно сделать

### US-002: Отметка задачи выполненной

**Как** пользователь  
**Я хочу** отмечать задачи как выполненные  
**Чтобы** видеть свой прогресс

### US-003: Удаление задачи

**Как** пользователь  
**Я хочу** удалять задачи  
**Чтобы** убирать ненужные или завершенные задачи

### US-004: Фильтрация задач

**Как** пользователь  
**Я хочу** фильтровать задачи (все/активные/завершенные)  
**Чтобы** фокусироваться на нужных задачах

### US-005: Сохранение данных

**Как** пользователь  
**Я хочу**, чтобы мои задачи сохранялись  
**Чтобы** не потерять их при закрытии браузера

## Key Features

1. Добавление задач с текстовым описанием
2. Отметка задач как выполненных/невыполненных
3. Удаление задач
4. Фильтрация: Все / Активные / Завершенные
5. Счетчик активных задач
6. Сохранение в localStorage
7. Адаптивный дизайн

## Non-Functional Requirements

- Быстрый отклик интерфейса (<100ms)
- Работа в современных браузерах (Chrome, Firefox, Safari, Edge)
- Адаптивность для мобильных устройств
- Доступность (WCAG 2.1 Level AA)

## Язык и локализация

- **Русский — основной язык приложения** (язык по умолчанию, fallback).
- Другие языки (английский и т.д.) — второстепенные, добавляются по необходимости.
- При создании или модификации кода учитывать: новые строки в UI сначала на русском.
