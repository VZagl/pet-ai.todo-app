# Memory Bank: Project Brief

## Project Name
pet.todo - TO-DO Application

## Project Description
Учебный проект TO-DO приложения на React для изучения современных практик разработки фронтенд приложений.

## Project Goals
1. Создать функциональное TO-DO приложение
2. Применить лучшие практики React разработки
3. Использовать TypeScript для типобезопасности
4. Покрыть код тестами
5. Обеспечить хорошую UX/UI

## Project Scope
- Управление списком задач (CRUD операции)
- Фильтрация и сортировка задач
- Персистентность данных через localStorage
- Адаптивный дизайн
- Покрытие тестами

## Project Constraints
- Использовать существующий стек технологий (React 19, Vite, TypeScript)
- Не использовать внешние UI библиотеки на начальном этапе
- Следовать установленным конвенциям проекта

## Success Criteria
- Все основные функции работают корректно
- Код покрыт тестами (>80%)
- Приложение адаптивно
- Код соответствует стандартам качества (ESLint, Prettier)
